# Calculadora Web

Uma calculadora desenvolvida com HTML, CSS e JavaScript puro com foco em prática de lógica de programação, manipulação de DOM e controle de estados.
---

## Funcionalidades
- Operações básicas:
  - Soma
  - Subtração
  - Multiplicação
  - Divisão
  - Potência
  - Porcentagem
- Recursos extras:
  + Alteração de sinal (+/-)
  + Memória (M+, M-, MR)
  + GT (Grand Total)
  + Tratamento de erros
  + Formatação automática do visor
  + Controle de múltiplos pontos decimais
  + Pop-ups informativos

---

## Tecnologías utilizadas
HTML5
CSS3
JavaScript

---

## Objetivo do projeto
O projeto foi desenvolvido com o objetivo de praticar:

- Manipulação de eventos
- Controle de estados
- Organização lógica de operações matemáticas
- Manipulação de strings e números
- Tratamento de erros
- Estruturação de interface com diferentes métodos visuais (grid, flexbox etc.).
