# Focus Time App ⏰

Aplicação web desenvolvida com HTML, CSS e JavaScript puro que reúne três ferramentas de gerenciamento de tempo em uma única interface:

- Relógio Digital
- Cronômetro
- Pomodoro

O projeto foi criado com o objetivo de praticar manipulação do DOM, eventos, temporizadores (`setInterval`), navegação entre páginas e estilização com CSS.

## Funcionalidades

### Relógio Digital
- Exibição da hora em tempo real
- Exibição da data atual
- Atualização automática dos valores

### Cronômetro
- Iniciar contagem
- Pausar contagem
- Reiniciar cronômetro
- Navegação para outras páginas da aplicação

### Pomodoro
- Modo foco
- Pausa curta
- Pausa longa
- Controle de iniciar e pausar
- Alternância automática entre os modos
- Fluxo contínuo de produtividade

## Conceitos Praticados 📚
